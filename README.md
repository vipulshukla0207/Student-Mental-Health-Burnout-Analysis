# Student-Mental-Health-Burnout-Analysis

Project Overview

This project analyzes student mental health and burnout levels using Python and data analysis libraries.
The goal is to identify key factors affecting student stress and burnout and derive meaningful insights.

Tools & Technologies Used--

-> Python 

-> NumPy

-> Pandas

-> Matplotlib

-> Seaborn

-> Google Colab

Dataset

-> Dataset: Student Mental Health & Burnout Dataset

-> Size: ~150,000 rows, 20 features

-> Features include:
  
  -> Age, Gender, Course
  
  -> Study Hours, Sleep Hours
  
  -> Stress Level, Anxiety Score, Depression Score
  
  -> CGPA, Attendance
  
  -> Burnout Level

Data Cleaning

-> Handled missing values

-> Converted data types (object → numeric where required)

-> Removed/handled outliers

-> Created new feature: burnout_category (Low / Medium / High)

Exploratory Data Analysis (EDA)

Key Visualizations:

-> Scatter plots (relationship analysis)

-> Box plots (category comparison)

-> Histogram (distribution analysis)

-> Line plot (trend using aggregation)

-> Pie chart (category distribution)

-> Heat map

Key Insights

Strong Relationships:

-> Sleep vs Stress → More sleep reduces stress

-> Study Hours vs Burnout → More study increases burnout

-> Majority of students fall in moderate stress.

-> Attendance vs CGPA -> Attendence alone is not a srong predictor of academic performance..

-> Screen time vs stress -> High screen time high stress

-> Sleep vs Burnout --> Less sleep, more burnout

-> Academic Pressure vs Stress --> Academic pressure increases stress level increases. 

-> Majority of students fall into medium burnout category

Distribution Insight:

-> Stress levels follow a normal distribution, with most students having moderate stress

Burnout Distribution:

-> Medium Burnout → ~50%
-> High Burnout → ~42%
-> Low Burnout → ~8%

Indicates burnout is widespread among students

Conclusion

-> Student burnout is a significant issue

-> Sleep and study habits are the most influential factors

-> Academic performance alone does not determine stress

-> Balanced lifestyle is crucial for mental well-being
